## What This Project Is

Design Canvas is a workspace/playground built on top of the `@david-richard/ds-blossom` design system. It exists to exercise, iterate on, and demonstrate DS components in a real React application context.

---

## Stack

- **Build:** Vite
- **Framework:** React + TypeScript
- **Styling:** Tailwind CSS v4
- **Design system:** `@david-richard/ds-blossom` (consumed as a package)
